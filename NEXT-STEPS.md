# AI Accelerator - Next Steps

Post-install checklist. Work through each item to complete your setup.

## 1. Review Generated Settings

The installer generated two settings files:

### .claude/settings.json (committed, shared)
Contains a baseline permissions whitelist for your detected tech stack. Review it and add project-specific patterns as needed. This file is committed to version control so your team shares the same baseline.

### .claude/settings.local.json (gitignored, secrets)
Contains placeholder `env` entries for tokens and secrets. Fill in your actual values. This file is NOT committed.

### Fix: Agent Teams Feature Flag

The installer detected that the agent teams flag is not set. Add to `~/.claude/settings.json`:
```json
{
  "env": {
    "CLAUDE_CODE_EXPERIMENTAL_AGENT_TEAMS": "1"
  },
  "preferences": {
    "teammateMode": "auto"
  }
}
```
Then restart Claude Code.

## 3. OpenSpec Setup

OpenSpec was selected but could not be initialized automatically. Set it up manually:

```bash
# Install OpenSpec (if not installed)
npm install -g openspec

# Initialize in your project root
openspec init --provider claude-code
```

This creates the `openspec/` directory and enables the `/opsx:*` slash commands in Claude Code.
